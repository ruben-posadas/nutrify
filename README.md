# nutrify
A full-stack web application that helps users track nutrition, plan meals, and generate cost-efficient grocery lists.

## Features (Planned)

- User authentication
- Calorie and macro tracking
- Meal planning
- Grocery list generation
- Cost estimation for meals

## Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: PostgreSQL (or SQLite)

## Purpose

This project is being built to demonstrate full-stack development skills for SWE/co-op roles.
